# Additional IBM i & Bob Labs: MCP, Bob Shell, Dashboard with Ansible

## Overview

These labs cover advanced customization of IBM Bob for IBM i — going beyond code modernization into **system integration, AI-powered tooling, and operational automation**. They are designed as standalone, self-contained labs that complement the core SAMCO modernization series.

You will configure Bob with IBM i-specific MCP tools and custom modes, query live IBM i systems using natural language, and automate system administration tasks such as PTF management using Bob-assisted Ansible playbooks.

For essential concepts and best practices when using IBM Bob for IBM i modernization, see the [Bob Concepts & Best Practices](./bob-concepts-and-best-practices.md) guide.
